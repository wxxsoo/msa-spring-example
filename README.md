# msa-spring-example
## 프로젝트 소개
Spring 기반 MSA 구축 기술 검토를 위한 예제 프로젝트입니다.
## 필수 설치 사항
프로젝트를 구동하기 위해 로컬 환경에 다음 요소들의 설치가 필요합니다.
- Java 11
- Elasticsearch
- Kibana
## 기술 스택
다음의 기술들을 사용해 MSA를 구성했습니다.
- Spring Boot 
- Spring Cloud Gateway
- Spring Cloud Eureka
- Spring Cloud Sleuth
- Zipkin
- Elasticsearch
- Kibana
